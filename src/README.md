# Source Directory Structure

This directory contains the source code for the North Pole Radar application.

## Directory Structure

```
src/
├── components/          # React components
│   ├── ChristmasBackground.jsx
│   ├── Decorations.jsx
│   ├── ElfWorkshop.jsx
│   ├── MusicPlayer.jsx
│   ├── Notifications.jsx
│   ├── ReindeerStatus.jsx
│   ├── SantaMap.jsx
│   ├── SantaTracker.jsx
│   ├── ShareButton.jsx
│   ├── Snow.jsx
│   └── ThemeToggle.jsx
├── styles/             # CSS stylesheets
│   ├── App.css
│   ├── ChristmasBackground.css
│   ├── Decorations.css
│   ├── index.css
│   └── Snow.css
├── utils/              # Utility functions (if needed)
├── App.jsx             # Main application component
├── main.jsx            # Application entry point
└── README.md           # This file
```

## Organization

- **components/**: All React components are organized here
- **styles/**: All CSS files are centralized in this directory
- **utils/**: Utility functions and helpers (if needed)
- **App.jsx**: Main application component that orchestrates the app
- **main.jsx**: Entry point that renders the React app
